# AI Agent Instructions for Coding-Practice

This repository contains simple Python practice files and notebooks. There is no formal build system, package manifest, or test framework.

## What agents should know

- The project is a collection of Python exercises, mostly `.py` scripts and `.ipynb` notebooks.
- Use Python 3-style idioms and keep changes small and easy to understand.
- Prefer standard library solutions; do not add external dependencies.
- Preserve the existing simple coding style and naming conventions.
- Do not create unnecessary infrastructure, frameworks, or project configuration files.

## How to edit

- Focus on fixing or improving the existing practice code in place.
- When adding new examples, place them alongside similar files in the root directory.
- Keep documentation minimal; only add files if the user explicitly asks for them.

## Special user preference

- Avoid opening or triggering an interactive agent window in VS Code while editing this repository.
- Use direct file updates and concise responses instead.
